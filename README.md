# Volant d'inertie — Stockage d'énergie mécanique (TIPE PSI)

## Description
Étude et optimisation d'un volant d'inertie destiné au stockage d'énergie mécanique,
réalisée dans le cadre du TIPE (Travail d'Initiative Personnelle Encadré) en CPGE filière PSI.

## Objectifs
- Modéliser le comportement dynamique d'un volant d'inertie
- Optimiser la géométrie pour maximiser l'énergie stockée sous contrainte de résistance des matériaux
- Simuler numériquement les résultats

## Contenu
- `rapport/` : rapport écrit du TIPE
- `code/` : scripts de simulation numérique

## Outils utilisés
- Python (NumPy, Matplotlib)
- Modélisation mathématique et simulation numérique
